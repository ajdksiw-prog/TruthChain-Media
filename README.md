# TruthChain-Media

## Overview

TruthChain-Media is a blockchain-based platform designed to combat misinformation and enhance media credibility through collaborative fact-checking, source verification, and journalist reputation systems. Built on the Stacks blockchain using Clarity smart contracts, this platform provides cryptographic verification of news sources, community-driven fact-checking, and incentive mechanisms for accurate reporting.

## Problem Statement

In today's digital age, misinformation spreads rapidly across media platforms, eroding public trust in journalism and creating confusion around factual information. Traditional fact-checking methods are often centralized, slow, and subject to bias. TruthChain-Media addresses these challenges by:

- Providing cryptographic verification of news sources and documents
- Creating transparent journalist credibility scoring systems
- Enabling community-driven fact-checking with expert validation
- Implementing token-based incentives for accurate reporting and verification

## Getting Started

### Prerequisites
- Clarinet CLI installed
- Node.js and npm
- Git version control
- Stacks wallet for testing

### Installation
```bash
# Clone the repository
git clone https://github.com/ajdksiw-prog/TruthChain-Media.git

# Navigate to project directory
cd TruthChain-Media

# Install dependencies
npm install

# Run tests
clarinet test

# Check contracts
clarinet check
```

### Testing Contracts
```bash
# Test source verification contract
clarinet test tests/source-authenticity-verification_test.ts

# Test journalist credibility contract
clarinet test tests/journalist-credibility-system_test.ts
```
